# Chutes & Levers Studio

A puzzle game engine and editor inspired by the original **Chutes & Levers** concept.

Chutes & Levers Studio is a complete toolkit for designing, playing, and solving grid-based puzzles built around gravity, flipping mechanics, and spatial reasoning.

---

## 🎮 What is this?

Chutes & Levers Studio combines three things into one project:

* **Game** – play handcrafted puzzles
* **Editor** – build your own puzzles visually
* **Solver** – validate and analyze solutions

The game is based on a two-sided card system where flipping the board is a core mechanic.

---

## 🧠 Core Mechanics

* **Walls** block movement
* **Levers** modify the state or routing
* **Chutes** transport the player across the board (often between sides)
* **Gravity** affects movement after every transition
* **Card flipping** changes the entire puzzle state

These systems interact to create surprisingly deep puzzles from simple rules.

---

## 🎯 Goal

Navigate from **START** to **GOAL**.

Sounds simple — but requires:

* planning across both sides of the card
* predicting gravity after flips
* understanding chute pair relationships

---

## 🛠 Features

* Visual puzzle editor
* Built-in solver (for validation and difficulty testing)
* JSON-based puzzle format
* Support for horizontal and vertical starting orientations
* Color-coded chute pairs for clarity
* Mobile-first design (iPad / iPhone compatible)
